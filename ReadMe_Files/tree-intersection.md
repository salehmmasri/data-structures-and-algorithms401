# Challenge
- Find common values in 2 binary trees.

___

## Approach & Efficiency
- Time Big O(n) where n is amount of elements in the biggest tree. Space Big O(n)

___

## Input:
- tree 1
```
             __10___
            /      \
         __7       _2
        /   \     /  \
       6     9   11   1
      / \
     5   8

```
- tree 2
```
                ___10___
               /        \
          ___  4        _5
         /      \      /  \
       _23       9    11   45
      /   \
     32    8
```
# Output: {10,9,8,11}
